# Manifesto Signatories

These community members have signed the I Know My LLM Manifesto, committing to trustworthy AI evaluation practices.

## How to Sign

### Option 1: Create a Pull Request (Recommended)

1. **Fork this repository**
2. **Edit the CSV file**: `public/signatures.csv`
3. **Add your signature** using this format:
   ```
   name,title,email,comment
   ```
4. **Submit a Pull Request**

#### Signature Fields

| Field | Required | Description | Anonymous Option |
|-------|----------|-------------|------------------|
| `name` | Yes | Your name or identifier | Use `Anonymous` or a pseudonym |
| `title` | No | Your role/position | Leave empty or use `-` |
| `email` | No | Contact email | Use `anonymous@example.com` or leave empty |
| `comment` | No | Why you're signing | Any text or leave empty |

#### Example Entries

```csv
# Full signature
Jane Doe,Security Researcher,jane@example.com,Evidence over promises - always.

# Anonymous with title
Anonymous,CTO,,Transparency matters for our industry.

# Minimal anonymous
Anonymous,,,

# Pseudonym
TrustVerifier,AI Ethics Advocate,trustverifier@proton.me,Community wisdom over isolated judgment.
```

### Option 2: Open an Issue

Can't create a PR? Open an issue with your signature details and we'll add you.
